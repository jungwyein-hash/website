## 4. Apple/Tesla 차용 패턴

- **톱 nav = 제품 패밀리 단위만**, 모델은 메가 패널 안에. (Tesla 이미지-led 메가 패널 차용)
- **제품 페이지 sticky 서브-nav** — Apple MacBook Pro 패턴 (개요/특장점/사양/시공/자료/문의)
- **빅 숫자 헤드라인 스펙** + 별도 사양표 (Tesla 감성 + Apple 데이터의 하이브리드)
- **Compare 페이지** — PO필름 grade를 옆으로 비교 (Apple compare)
- **멀티칼럼 푸터** — 제품군/회사/자료실/문의/채용

### 차용 안 함
- "Buy Now" 같은 e-com CTA → **`견적 요청` / `대리점 찾기` / `자료 다운로드`** 로 치환
- Tesla 비디오 히어로 → 풍경 사진/시공 사진으로
- Apple 3D 뷰어 → 과한 기술. 스킵
- 라이프스타일 모델 사진 → **농가/현장/공장 다큐멘터리 사진**

## 5. Aesop/Bluebottle 차용 톤

- **카드 사용 X.** 이미지 + 이름 + 한 줄 스펙 + 1px 헤어라인. 여백이 분리자.
- **헤더 슬림** (~64px), 메가-메뉴 대신 풀-페이지 오버레이도 가능
- **푸터 무겁게 에디토리얼** (4컬럼 + 뉴스레터)
- **편집형 단락** 60ch, 좌측 정렬
- 모션 무절제 X. 호버는 200ms 라인 그로우, 이미지 0.98→1.0 + 밝기 +2%
- 사진은 차분하고 약간 쿨한 톤. 골든아워, 농민의 손, 하우스 내부 빛, 정물 (브랜드북도 동일 방향)
